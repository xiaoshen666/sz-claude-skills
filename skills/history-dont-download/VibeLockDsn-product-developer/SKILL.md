# Product Developer

## Workflow:
After capturing the user's development requirements, execute according to the following steps. After completing each step, confirm with the user (provide three options: 1. Feasible and continue; 2. Need modification, please explain; 3. User modifies in code). Choose 1 to continue, choose 2 to modify based on user feedback, choose 3 to view modification content.

### Step 1: Read Product Design Documents
- Read the output of `VibeLockDsn-product-manager` skill (PRD document)
- Parse PRD document structure and extract key information

### Step 2: Technology Stack Confirmation
- Extract recommended technology stack information from PRD
- Show recommended technology stack options to the user and confirm final selection

### Step 3: Development Environment Preparation
- Configure development environment according to confirmed technology stack requirements
- Create project structure and basic configuration files

### Step 4: Full-stack Development Implementation
- **Database Design and Implementation**: Design table structure, create migration scripts, generate demonstration data
- **Backend Development**: Create API interfaces, implement business logic, set up authentication and authorization
- **Frontend Development**: Create components, integrate APIs, implement interactive logic

### Step 5: Self-testing Based on Test Cases
- Read and execute comprehensive testing on test case documents generated by `VibeLockDsn-product-manager` skill
- Strictly execute unit testing, integration testing, and end-to-end testing according to test cases
- Record test results and identified defects

### Step 6: Testing Verification
- Write supplementary test cases and execute automated testing
- Verify functional completeness and performance requirements

### Step 7: Build and Deployment
- Code packaging and optimization
- Generate deployment configuration and CI/CD configuration

### Step 8: Environment Startup and Verification
- Automatically start development environment and verify service status
- Fault handling and repair verification

## Technology Stack Support:
- **Frontend Frameworks**: React (recommended), Vue, Angular, Next.js, Nuxt.js
- **Backend Frameworks**: Python + FastAPI (recommended), Express.js, NestJS, Django, Flask, Spring Boot
- **Databases**: MySQL (recommended), PostgreSQL, MongoDB, Redis, SQLite
- **Styling Solutions**: CSS, SCSS, Tailwind CSS, Styled Components, ShadCN UI
- **State Management**: Redux, Vuex, MobX, Context API
- **Build Tools**: Webpack, Vite, Rollup
- **Testing Frameworks**: Jest, Cypress, Playwright, Junit, pytest
- **Package Management**: npm, yarn, pnpm, pip, gradle, maven
- **DevOps Tools**: Docker, Kubernetes, CI/CD (GitHub Actions, Jenkins)
