# CDS 企业管理系统 设计风格文档

## 1. 整体布局

- **布局类型**：内容区域全宽布局，无侧边栏（截图中未出现全局侧边栏，仅局部页面有树形分类侧边栏）
- **页面结构**：顶部标题栏 + 工具栏区域 + 搜索筛选区 + 内容主体区（表格/表单）+ 底部分页器
- **内容区背景**：浅灰色 `#F5F5F5` / `#F0F2F5`
- **表单/表格主体背景**：纯白色 `#FFFFFF`

## 3. UI元素

### 3.1 顶部标题栏（PageHeader）
- 高度：约 40px
- 背景色：`#1677FF`
- 文字颜色：`#FFFFFF`
- 字号：14px，字重 500
- 内容：页面标题 + 操作人信息（灰白色，字号12px）+ 右侧辅助按钮（流程图、审批记录）
- 左上角有当前环节状态文字（右对齐）

### 3.2 工具栏区域（Toolbar）
- 背景色：`#FFFFFF`
- 高度：约 48px，padding：8px 16px
- 包含：页面标题（左侧竖线标识） + 操作按钮组（新增、发起工作流等）
- 标题左侧有蓝色竖线装饰 `#1677FF`，宽3px，高约14px

### 3.3 搜索筛选区（SearchBar）
- 背景色：`#FFFFFF`
- padding：16px
- 底部有 `#E8E8E8` 分割线
- 布局：多列栅格（3列），每列包含：标签 + 条件下拉（包含/等于/范围） + 输入框
- 右侧：**查询**按钮（主色实心）+ **清空**按钮（文字按钮）+ 折叠图标按钮

### 3.4 卡片容器（Card）
- 背景色：`#FFFFFF`
- 圆角：4px 或无圆角
- 边框：无（依靠背景与页面背景色区分）
- 阴影：无明显阴影

### 3.5 数据表格（DataTable）
- 表头背景：`#FAFAFA`
- 表头文字：`#333333`，字号13px，字重500
- 表头底部边框：`#E8E8E8`，1px solid
- 行高度：约 44px
- 行分割线颜色：`#F0F0F0`，1px solid
- 奇偶行：无交替色（全白）
- 悬停行背景：`#F0F7FF`（淡蓝色）
- 序号列宽度：约 50px
- 操作列：文字按钮，修改=`#1677FF`，删除=`#FF4D4F`，间距 8px
- 右上角有列设置图标（⚙）

### 3.6 分页器（Pagination）
- 位于表格右下方
- 文字："共 N 条 第 X/Y 页"
- 页码按钮：方形，当前页高亮（`#1677FF`边框+文字）
- 每页条数下拉：右侧 "20条/页 ∨"

### 3.7 输入框（Input）
- 高度：32px
- 圆角：4px
- 边框：1px solid `#D9D9D9`
- placeholder颜色：`#BFBFBF`
- 激活边框：`#1677FF`
- 背景色：`#FFFFFF`（禁用/只读：`#F5F5F5`）
- 字号：14px

### 3.8 下拉选择框（Select）
- 样式同输入框，右侧有 ∨ 箭头
- 带搜索图标的选择框（关联搜索）：右侧有 🔍 图标，颜色 `#1677FF`

### 3.9 条件选择下拉（小尺寸，前置于输入框）
- 宽度：约 70-90px
- 高度：32px
- 背景：`#FFFFFF`
- 边框：1px solid `#D9D9D9`，圆角4px
- 文字右侧有 ∨ 图标

### 3.10 按钮

#### 主按钮（PrimaryButton）
- 背景：`#1677FF`
- 文字：`#FFFFFF`
- 高度：32px，padding：0 16px
- 圆角：4px
- 字号：14px

#### 次要按钮（SecondaryButton / "保存"）
- 背景：`#1677FF`（保存按钮与主按钮相同蓝色）
- 文字：`#FFFFFF`
- 部分场景：白色背景 + `#D9D9D9` 边框 + `#333333` 文字

#### 文字按钮（TextButton）
- 无背景无边框
- 操作类（关闭）：`#666666`
- 操作类（清空）：`#666666`

#### 危险按钮（DangerButton / "驳回"）
- 背景：`#FF4D4F`
- 文字：`#FFFFFF`
- 高度：32px，圆角4px

#### 图标按钮（IconButton）
- 无背景，图标颜色 `#666666`
- 悬停颜色 `#1677FF`

### 3.11 开关组件（Switch）
- 关闭状态：灰色背景 `#BFBFBF`，圆形滑块
- 开启状态：`#1677FF` 背景
- 尺寸：宽约 40px，高约 22px

### 3.12 对话框（Modal）
- 背景：`#FFFFFF`
- 圆角：8px
- 标题栏：白色背景，标题文字 `#333333`，字重600，字号16px
- 关闭按钮：右上角 ×，`#999999`
- 底部操作区：右对齐，按钮间距8px
- 遮罩：`rgba(0,0,0,0.45)`

### 3.13 消息提示（Toast）
- 成功提示：绿色圆形✓图标 + 白色卡片 + 描述文字
- 中间居上浮层弹出，圆角8px
- 成功图标颜色：`#1677FF`（蓝色勾选）或 `#52C41A`

### 3.14 通知提示（顶部Toast）
- 位于页面顶部居中
- 圆角胶囊形，有 ✓ 图标，绿色
- 文字：`#333333`，字号14px

### 3.15 树形组件侧边栏（TreeSidebar）
- 宽度：约 220px
- 背景：`#FFFFFF`
- 标题：蓝色 `#1677FF` 文字 + 右侧 + 图标
- 搜索框：高度28px，灰色边框
- 树节点：左侧有折叠/展开图标，激活项高亮

### 3.16 日期选择器（DatePicker）
- 弹出双月历：左右两个月
- 背景白色，表头蓝色文字 `#1677FF`
- 当日期高亮：蓝色圆形背景 `#1677FF`，白字
- 表头：一二三四五六日，灰色 `#999999`
- 边框：1px solid `#D9D9D9`

### 3.17 文件上传组件（Upload）
- 虚线边框：`#D9D9D9`，圆角4px
- 上传图标：蓝色文档图标
- 提示文字：`#999999`
- 已上传文件列表：文件名+上传人+时间+操作（下载 `#1677FF` / 删除 `#FF4D4F`）

### 3.18 颜色选择器（ColorPicker）
- 色板区域
- 下方色相条 + 透明度条
- HEX输入框 + 透明度百分比输入框
- 底部预览色块

## 4. 排版

- **字体**：系统默认字体（无特殊自定义字体）
- **基础字号**：14px
- **页面标题**：14px，`#FFFFFF`
- **卡片标题**：14px，`#333333`，字重500
- **表单标签**：14px，`#333333`
- **占位文字**：14px，`#BFBFBF`
- **表格内容**：13-14px，`#333333`
- **辅助文字**：12px，`#666666`
- **操作链接**：14px，`#1677FF` 或 `#FF4D4F`
- **行高**：约22px（正文），44px（表格行）

---

## 5. 交互元素

- **悬停效果**：按钮悬停加深，行悬停淡蓝背景
- **激活输入框**：边框变为 `#1677FF`，蓝色发光
- **加载状态**：按钮内旋转图标
- **成功反馈**：中央蓝色✓圆形Toast，自动消失
- **条件选择器**：输入框前置小型下拉（包含/等于/范围）
- **折叠筛选区**：右上角折叠图标，可收起搜索条件

---

## 6. 空状态（EmptyState）
- 居中显示 3D 蓝色纸箱图标（立体感，带纸飞机）
- 文字："暂无内容"，`#999999`，14px
- 出现于表格无数据时

---

## 7. 流程相关组件

### 审批记录列表（ApprovalLog）
- 左侧蓝色圆点 `#1677FF`
- 步骤名称：`#333333`，字重500
- 时间：右对齐，`#999999`
- 操作人、审批结果、备注：`#666666`

### 流程按钮组（WorkflowButtons）
- 底部固定区域："处理结果：" + 提交（主色蓝）+ 驳回原因下拉（次要）+ 驳回（红色）+ 保存（蓝色）
- 意见填写区：宽输入框 + 右侧"常用意见 ∨"

---

## 8. 核心设计原则

1. 极简蓝白风格，主色 `#1677FF`
2. 搜索区与表格区之间有明确视觉分层
3. 操作按钮使用颜色语义区分：蓝色=常规操作，红色=危险操作
4. 所有交互组件边角为4px圆角
5. 图标简洁线条风格，无填充
