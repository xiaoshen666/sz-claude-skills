创建三个skill
cds-product-design-zh: 根据需求概设整理需求输出需求详设 包括根据已有UI库生成UI稿 包括前端架构师生成前端技术实现方案（基于技术选型框架已经确定 目前结构已经确定） 包括后端架构师生成后端技术实现方案（基于技术选型框架已经确定，目录结构已经确定）
cds-product-develop-zh:依据已有的UI稿和前端技术实现方案生成前端代码能力，依据已有的后端技术实现方案实现后端代码能力 生成启动类代码能力 包含git提交管理方便回退代码与模块集成能力
cds-product-test-zh:依据已有的代码结构编译启动前端资源，依据已有的后端代码结构编译启动后端项目 需要配置 已经安装了codelink的环境信息或者（nacos地址 数据库地址 国际化服务地址 如有需要redis地址 kafka地址） 
cds-product-devops-zh:依据已有的项目结构 编译构建 通过之后打包代码 压缩之后调用cds接口上传 成功之后  触发远程构建安装启动 webhook方式通知企微机器人
